# **App Name**: IT Horizon

## Core Features:

- Post Creation: Users can create and submit posts with formatted text, images, and code snippets.
- Categorization & Tagging: Posts can be categorized into predefined topics (e.g., News, Programming, Design) and tagged with user-defined tags for filtering and search.
- Question & Answer Forum: Dedicated section for users to ask questions and receive answers from the community.
- Detailed Post and Question Pages: Individual pages for posts and questions with integrated comment sections and answer submission forms, as well as functional internal and external links to third-party sites and pages.
- Search & Filtering: Robust search functionality and filters to sort posts/questions by topic, tags, date, and popularity.
- Pagination: Implement pagination to efficiently handle large volumes of content and improve site navigation.

## Style Guidelines:

- Primary color: Soft, muted blue (#94A9FF) to create a calm, futuristic feel without being overly vibrant.
- Background color: Light gray (#F0F2F5), close in hue to the primary, creates a clean, spacious backdrop similar to StackOverflow or Habr.
- Accent color: Pale violet (#DABFFF) for interactive elements to draw attention, ensuring a futuristic aesthetic.
- Font: 'Inter' (sans-serif) for both body text and headlines. It's modern, readable, and suitable for technical content. Note: currently only Google Fonts are supported.
- Use simple, line-based icons for categories and tags to maintain a lightweight and clean visual style.
- Prioritize a clean, well-spaced layout with a focus on readability and easy navigation, similar to popular tech forums.
- Subtle animations for loading new content and transitions between pages to add a touch of modern, smooth user experience.
